Examples

You can add a line to the XAML as follows:

Or in C#:

HtmlCanvas myCanvas = new HtmlCanvas() { Width = 1000, Height = 500 };
LineElement myLine = new LineElement()
{
    X1 = 200,
    Y1 = 42,
    X2 = 100,
    Y2 = 50,
    StrokeColor = Colors.Red,
    LineWidth = 2
};
myCanvas.Children.Add(myLine);
