Class HtmlCanvasElement

Defines an element which can be in a HtmlCanvas. It cannot be instantiated, but it provides some common properties. Every HtmlCanvas child is an HtmlCanvasElement. Full documentation is available at: http://cshtml5.com/links/how-to-use-the-html5-canvas.aspx
